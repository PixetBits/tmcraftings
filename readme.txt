This mod replaces all default ores (except coal), adds new tools and replaces the
default ones (wooden and stone pickaxe/shovel). Replace some textures, adds a new
chest (Iron Crate), Furnace (Magic Furnace) and a redstone-like system.

This mod changes the default inventory, furnaces and chests GUIs (replacing the
last two).

And more!
